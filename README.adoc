= Kawa web collection
:toc: left

== Prerequisites

* Java 11

* Maven 3

== Installation

Run `mvn install` from the root of this repository

== Generating a project

To generate a scaffold project, execute

```
mvn archetype:generate -DarchetypeGroupId=com.github.arvyy.kawa-web-collection
                       -DarchetypeArtifactId=auth-archetype
                       -DarchetypeVersion=0.0.1
```
